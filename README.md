# HI Leads

Triage tool for Higney International high-quality leads under the Maggiano's framework.

Browse: filter by tier / category / status, expand a card for full notes, expansion timeline, kill risks, contact, outreach draft.

Edit: update `leads.json` and push. The site refetches on reload.

When a lead goes active in HubSpot, mark `status: "active"` here and tell Artemis directly to push to HubSpot.
